## Tripd (A01)
The **Tripd** app is a road-trip planner for Android. Users can create a trip by adding *start* and *end* dates, and add **events** to the days of their trip.

The **Tripd** app has four pages:
* Review Trips
* Trip & Event Info
* Add/Edit Trip
* Add/Edit Event
